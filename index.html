📖 How IndexFlow Works

IndexFlow uses Google's official Indexing API to notify Google about your pages.

1. Submit URLs
Paste your URLs in the Submit tab — one per line. Up to 200 per task.
2. API Notification
IndexFlow calls Google Indexing API for each URL using your service account credentials.
3. Google Crawls
Google receives the notification and crawls/indexes your page — typically within 24 hours.
⚠️ Important Note
For best results, add the service account email as Owner in Google Search Console for each property you want to index.
